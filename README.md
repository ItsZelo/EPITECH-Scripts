# EPITECH-Scripts
A bunch of useful scripts for EPITECH students

reposcript :
USAGE : repository [option] [parameters]
Available options are:

	list
	push    [commit_name]
	clone   [repository]
	create  [repository]
	delete  [repository]
	status  [repository]
	deluser [repository] [user]
	adduser [repository] [user] [permissions]
  
  Don't hesitate to report bugs :)
  Enjoy !
